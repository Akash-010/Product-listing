#  Product Listing Page

A responsive **Product Listing Web App** built with **Next.js**, **Tailwind CSS**, and  featuring **Server-Side Rendering (SSR)**, live product data from a public API, a **search bar**, and optional **pagination** and **product detail pages**.

---

##  Tech Stack

- **Next.js** (App Router with SSR)
- **React.js**
- **Tailwind CSS** for custom utility-first styling
- **Axios** for data fetching
- **FakeStore API** - https://fakestoreapi.com

---

##  Features

-  **Server-Side Rendering**: Products are fetched on the server and rendered before the page is delivered.
-  **Search Functionality**: Filter products by name.
-  **Product Cards**: Show title, image, and price.
-  **Product Details Page** *(optional)*: Navigate to see product details by ID.
-  **Pagination** *(optional)*: Display products page by page.
